# grocery_app

A new Flutter project.


## Setup

1. **Prerequisites**: Ensure you have Flutter installed.
2. **Environment Variables**:
   - Copy `.env.example` to `.env`.
   - Update `SUPABASE_URL` and `SUPABASE_ANON_KEY` with your actual Supabase project credentials.
3. **Dependencies**: Run `flutter pub get`.
4. **Run**: Use `flutter run` to start the application.
